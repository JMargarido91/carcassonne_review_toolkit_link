# Carcassonne Review Toolkit

A lightweight browser tool to review and analyze **2-player Carcassonne** matches played on **Board Game Arena**.

This tool enhances replays by providing detailed tile information, game-state insights, and real-time counters to support post-game analysis and strategic study.

---

## Requirements

This tool is designed specifically for:

- **Carcassonne (Base Game)**
- **2-player matches**
- **Board Game Arena (BGA)** only
- **Replay mode**
  - ‚ö†Ô∏è _Not available during live games_

---

## Features

### üî¢ Remaining Tiles Counter

See exactly how many tiles of each type are still left in the draw pile.

### üé¥ Tile Draw Tracking

Displays which tiles were drawn by each player throughout the match.

### üß© Tile Fit Guide

Shows:

- Which tiles can be placed in each available board location
- How many of each type are still available
- Total number of tiles for each board location

### üìä Live Score Tracker

Automatically updates as the replay progresses, giving a clear breakdown of the current board state.
