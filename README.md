# Carcassonne Review Toolkit

A lightweight browser tool to review and analyze **2-player Carcassonne** matches played on **Board Game Arena**.

This tool enhances replays by providing detailed tile information, game-state insights, and real-time counters to support post-game analysis and strategic study.

---

## Requirements

This tool is designed specifically for:

- **Carcassonne (Base Game)**
- **2-player matches**
- **Board Game Arena (BGA)** only
- **Replay mode**
